# 🎮 Game Library API

REST API для каталога игр на Go + PostgreSQL

## Стек
- Go 1.22
- PostgreSQL 16
- Docker

## Запуск

Копируем env файл:
```bash
cp .env.example .env
```

Запускаем:
```bash
docker compose up
```

API доступно на `http://localhost:8080`
